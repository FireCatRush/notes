**Table 2.7 Operation on Sequences :

| Operation       | Description                                |
| --------------- | ------------------------------------------ |
| s + r           | Concatenation                              |
| s \* n, n \* s  | Makes n copies of s, where n is an integer |
| s\[i\]          | Indexing                                   |
| s\[i:j\]        | Slicing                                    |
| s\[i:j:stride\] | Extended slicing                           |
| len(s)          | Length                                     |
